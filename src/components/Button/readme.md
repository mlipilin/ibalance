## Usage

```javascript
<Button block disabled processing size="xl" type="primary">
    Primary Button
</Button>
```

Left icon:

```javascript
<Button iconLeft={<Icon name="arrow-left" />}>Prev</Button>
```

Right icon:

```javascript
<Button iconLeft={<Icon name="arrow-right" />}>Next</Button>
```
